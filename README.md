# second-bookstore

My first Vue.js project, trying to complete an online second bookstore.

This project was initialized in late May, 2018. 

### Build Setup
```bash

$ npm install -g vue-cli

$ vue init webpack second-bookstore

$ cd second-bookstore

$ npm install

$ npm run dev
```
